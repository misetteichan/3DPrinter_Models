## stackchan_metal_case_takao_version

- ミニルーター等があると後加工が少し楽になるかも？（怪我に注意です。）
- サーボはメタルギアのものでないと動かないと思います。
- 薄い部分があるので、レビュー時にリスクを許容するか確認されます。ご自身で決めて返信してください。<br>英語でメールが来ます。
